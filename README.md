This AI will recognize cars and there brands

## Table of content

## Installation

To run this app, you'll need these libraries : 

- OpenCv : 
```sh
pip install opencv-python
```

- Numpy :
```sh
pip install numpy
```

## Execution

Run the script with the following command :

```sh
python pisteTaVoiture.py
```
